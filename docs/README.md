# JewelloSoft Official Documentation

Welcome to the official documentation for **JewelloSoft**, the modern, efficient, and comprehensive desktop solution designed specifically for jewelry retail businesses.

This documentation serves as a complete guide for shop owners, administrators, and sales staff to master every feature of the software—from first-time setup to daily billing operations and inventory management.

## Table of Contents

Explore the guides below to get started and learn how to run your business smoothly using JewelloSoft.

### [1. Getting Started](./user_guide/01-getting-started.md)
Learn the basics of JewelloSoft, including first-time launch, shop registration, and a complete breakdown of the user interface.

### [2. Daily Operations](./user_guide/02-daily-operations.md)
Discover how to manage your daily tasks, including setting daily gold and silver rates, updating making charges, and understanding the main dashboard statistics.

### [3. Billing & Invoicing](./user_guide/03-billing-and-invoicing.md)
A comprehensive guide to creating estimates and tax invoices. Learn how to add items, calculate weights, apply taxes, deduct old metal values, and print professional A4 receipts.

### [4. Inventory Management](./user_guide/04-inventory-management.md)
Keep your shop's stock organized. Learn how to add new jewelry items, categorize stock, track weights, assign HUIDs, and monitor low-stock levels.

### [5. Customer Management](./user_guide/05-customer-management.md)
Build strong relationships with your customers. This guide covers adding customer profiles, storing contact details, and viewing past purchase histories.

### [6. Settings & Preferences](./user_guide/06-settings-and-preferences.md)
Personalize your software experience. Update your shop profile, upload a watermark logo, configure GST details, and change the application's visual theme.

---

**Note to Staff & Cashiers:**
This software is designed to be fully functional offline, ensuring your shop operations are never interrupted by internet outages. Regular data backups occur automatically when an internet connection is established.
